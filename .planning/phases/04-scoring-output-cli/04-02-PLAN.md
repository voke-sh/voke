Wire the full CLI: a runLint orchestrator (resolveTarget -> ingest(live|file) -> runRules(createDefaultRegistry())

CLI build target so `voke` is an executable.

existing module (ingestion, engine, report builder, scoring) and the Plan 01 leaf modules; it reimplements nothing.

Output: cli/run-lint.ts, cli/program.ts, cli/index.ts (bin), tsup.config.ts, package.json bin field, and a
run-lint orchestration test against the offline Apideck fixture.
