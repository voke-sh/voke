# MTQS Versions

The MCP Tool Quality Specification (MTQS) is versioned. All published versions remain immutable and independently citable. Each version link below is permanent.

## Published Versions

| Version | Status | Published | Spec |
|---------|--------|-----------|------|
| v0.1 | Current | 2026-06-13 | [MTQS v0.1](/spec/v0.1/MTQS-v0.1) |

## Versioning Policy

- Old versions are **never deleted** or modified. Once published, a version is immutable.
- Each version is independently citable at its permanent URL (e.g. `/spec/v0.1/MTQS-v0.1`).
- Future versions will be added alongside existing versions.

## Contributing

Contributions to the spec are welcome. See [CONTRIBUTING.md](https://github.com/voke-sh/voke/blob/main/CONTRIBUTING.md) for the rule PR process, which requires primary-source citations and positive + negative test fixtures.
